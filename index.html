1. Intro to C#

1.01 The Name of the Computer

Program 01 prints the NetBIOS name of the local computer to the console. Some basic concepts and terms of C# programming are introroduced.

/*Program 01. 
prints the NetBIOS name of the local 
computer to the console.*/
using System;  
namespace ComputerStuff 
{    
  class Computer
  {    
    // Main method 
    static void Main(string[] args) 
    { 
      // Printing something
      Console.WriteLine("The name of this computer is: " + Environment.MachineName); 
      Console.ReadKey(); 
    } 
   } 
 }
/*Program 01.
Prints the name of a computer to the console*/ - a multi lines comment (lines 1 - 3)
using System - enables to include the System namespace in the program - the collection of fundamental classes of C# (line 4).
namespace ComputerStuff - declaration of the namespace ComputerStuff that contains one class Computer (line 5).
// Main method - a single line comment (line 9).
class - user-defined blueprint used to create objects. In this program no objects are created. Here, name of the class is Computer (line 6).
Main() - a method that is the entry-point a C# application (here the program control starts and ends) (line 10).
void Main() - the method Main() does not return anything (line 10).
static void Main() - the keyword static is a modifier that declares that the method Main() is a static member. The class to which this member belongs does not have to be instantiated (line 10).
Console.WriteLine() - a method of the Console class, defined in the System namespace that prints a message to the console. (line 13).
Environment.MachineName - a property of the System namespace that gets the NetBIOS name of the local computer(line 13).
Console.ReadKey() - a method of the Console class, defined in the System namespace that is used to halt a program execution until the user presses a key (line 14).
In C# we concatenate two strings using operator +:
"The name of this computer is: " + Environment.MachineName.

1.02 The Number Pi

Program 02 calculates the value of nuber Pi. Variables and data types in C#.

/* Program 02. 
calculates the value of number pi
pi = 4/1 - 4/3 + 4/5 - 4/7 ...
*/

using System;  
namespace SomeMath 
{    
  class NumberPi
  {    
     static void Main(string[] args) 
    { 
     //declaration and initialization of the variable pi
     double pi = 0.0; 

     //Test 1
     pi = 4/1 - 4/3 + 4/5 - 4/7 + 4/9;
     Console.WriteLine("Test 1: pi = " + pi);
    
     //Test 2
     pi = 0.0;
     pi = 4.0/1 - 4.0/3 + 4.0/5 - 4.0/7 + 4.0/9;
     
     int test = 2;
     string output = "Test " + test.ToString() + ": pi = " + pi.ToString();
     Console.WriteLine(output);
     
     //Test 3
     pi = 0.0;
     double num = 4.0; 
     int den = 1;
     
     pi = num / den;
     
     den = den + 2;
     pi = pi - num / den;  
     
     den = den + 2;
     pi = pi + (num / den);
     
     den += 2;
     pi -= num / den;  
     
     den += 2;
     pi += num / den;
     
     test = 3;
     Console.WriteLine("Test {0}: pi = {1}", test, pi);
    } 
   } 
 }
/*Program 01.
Outputs the name of a computer on the console*/ - multi lines comment (lines 1 - 2)
// Main function - single line comment (line 8).
class - The class contains the data and methods to be used in the program. Methods define the behavior of the class (line 6).
Main() - method is the entry-point of our application (line 9).
